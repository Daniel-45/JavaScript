JavaScript Eventos

Los eventos son funciones que se ejecutan cuando sucede algo

Ejemplos

  • El usuario elimina algo
  • El usuario mueve el ratón
  • El usuario redimensiona una ventana
  • El usuario escribe algo en un input
  • El usuario hace clic en algún elemento
